Quick Reference

Navigation

  • h j k l move cursor
  • w b word forward/back
  • 0 $ line start/end
  • gg G file start/end

Editing

  • i a insert/append
  • x dd delete char/line
  • yy p copy/paste
  • u undo

Search

  • /text search forward
  • ?text search backward
  • n N next/prev match
  • :%s/a/b/g replace all

Files

  • :w save
  • :q quit
  • :wq save & quit
  • :q! force quit
